# Tumbler

[![CI](https://github.com/brocla/rust-tumbler/actions/workflows/ci.yml/badge.svg)](https://github.com/brocla/rust-tumbler/actions/workflows/ci.yml)

A personal PDF viewer, editor, and printer for Windows, built with Tauri v2 (Rust backend, React/TypeScript frontend) and pdfium.

**Work in progress.**
